术语表

站点中常用的 vLLM 术语和缩写，后续填充内容时统一使用相同写法。

核心运行

  • EngineCore：vLLM V1 内部主循环。
  • Scheduler：决定每一步执行哪些请求的组件。
  • Executor / Worker：执行模型 forward 的进程。
  • ModelRunner：在 worker 中负责模型 forward 的对象。

内存与缓存

  • KV Cache：以 block 为单位管理的 KV 内存。
  • PagedAttention：基于分页 KV 的 attention kernel。
  • Block Table：逻辑 token 序列到物理 block 的映射。
  • Prefix Caching：复用相同前缀的 KV block。

分布式

  • TP：Tensor Parallel，张量切分。
  • PP：Pipeline Parallel，层切分。
  • DP：Data Parallel，请求切分。
  • EP：Expert Parallel，MoE 专家切分。
  • CP / DCP / PCP：Context Parallel 系列。

解码

  • Speculative Decoding：draft + verify 的解码策略。
  • Draft Model：投机解码中生成候选 token 的小模型。
  • Rejection Sampler：验证候选 token 并接受/拒绝。
  • Chunked Prefill：长 prompt 拆分到多步调度。
